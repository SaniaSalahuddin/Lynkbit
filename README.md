# Lynkbit

Lynkbit is a modern link-in-bio platform built with Next.js and MongoDB.  
It allows users to create personalized profile pages with unique handles and share all their important links in one place.

---

## 🚀 Features

- Unique handle/username system
- Real-time handle availability checking
- Personalized link pages
- MongoDB database integration
- REST API support with Next.js Route Handlers
- Fast and responsive UI

---

## 🛠️ Tech Stack

- Next.js
- MongoDB
- Node.js
- JavaScript
